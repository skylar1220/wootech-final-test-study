### 일반 객체 버전

```java
public Name input() {
    printer.printLine("");
    String name = reader.readLine();
    validator.validate(name, "");
    return Name.from(name);
}
```


### 리스트 버전
```java
public Names input() {
    printer.printLine("");
    String names = reader.readLine();
    validator.validate(names, "이름");
    return Names.from(Converter.splitToList(SEPARATOR, names));
}
```
