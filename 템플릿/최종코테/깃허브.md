1. Fork 후에 skylar1220/프로젝트이름을 local로 clone하기

```java
git clone https://github.com/skylar1220/java-baseball.git

// clone한 폴더로 이동하는 방법: cd {저장소 아이디}
cd java-baseball
```

2. 브랜치 생성 및 전환

4. git add & commit
```java
// 작업한 파일 목록 확인하기
git status

// 모든 파일 스테이징
git add .

// 확인 후 스테이징
git add -p

// 스테이징된 파일 전부 커밋하기
git commit -m "feat : 기능 잘 구현"
```

5. git push
```java
git push origin skylar1220
```
